starUML3.0-mac破解

  • 从官网下载3.X的版本：https://staruml.io/download

  • 本文采用版本：StarUML-3.2.2.dmg

破解过程

  1. 下载并安装StarUML

  2. 安装npm ：brew install node

  3. 用npm安装asar ：npm install asar -g (-g表示全局安装)

    1. 如果安装失败，可能是镜像问题，建议安装国内镜像源。

  4. 解压文件StarUML中的app.asar，修改lisence文件

  1. 打包且覆盖原来的app.asar：asar pack app app.asar

文件打不开问题 — “App” 已损坏，无法打开。 您应该将它移到废纸篓。

当前Mac 版本为： Mac Big Sur 11.3.1

  1. 打开终端 Terminal；
  2. 输入以下命令修复指定 app，其中 xxx ：sudo xattr -r -d com.apple.quarantine xxx
  3. 最后需要你输入自己的密码授权。
