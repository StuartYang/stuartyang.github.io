Java语言的的特性

  • Java语言简单：语法与 C 语言和 C++ 语言接近。
    • Java 语言不使用指针，而是引用。
    • JVM提供了自动分配和回收内存空间，使得程序员不必为内存管理而担忧。
  • Java 语言是面向对象的
    • 封装：类将对象的属性和方法结合为一个独立的整体，对外并尽可能隐藏对象的内部实现细节。
    • 继承：支持类之间的单继承，但支持接口之间的多继承
    • 多态：
      • 重写父方法
        • 参数列表必须完全相同
        • 返回类型可以不相同，但是必须是父类返回值的派生类
        • 访问权限不能比父类中被重写的方法的访问权限更低
        • final 的方法不能被重写
        • 使用 super调用父类的被重写方法
      • 方法重载：
        • 参数不同
          • 个数不同
          • 顺序不同
          • 类型不同
        • 返回类型可以不同
        • 访问修饰符可以不同

java的平台无关性

javac xxx.java ： Java文件编译成class文件

java xxx.class 执行class文件字节码

javp -c xxx.class：反汇编class文件字节码

  1. 不用每次都需要做编译前的准备工作
  2. 兼容性：一次执行，多处运行
